# Um-Junaib


A CLI-based Snake game written in Go.

## Description

A CLI-based Snake game built with Go. ⚠️ Warning: this snake has a tendency to sidewind!

## Build & Run


### Build
```bash
go build -o um-junaib
```

### Run
```bash
go run .
```


## Controls
- Arrow keys: Move snake
- Any key: Start/Restart game
- Q or ESC: Quit
